HDD Scheduling Simulator
An interactive web application to visualize and compare disk scheduling algorithms.

🚀 Features
- Supports FCFS, SSTF, SCAN, C-SCAN, LOOK
- Step-by-step disk head movement animation
- Seek time calculation
- Algorithm comparison
- Clean and modern UI

🧠 Algorithms Implemented
- First Come First Serve (FCFS)
- Shortest Seek Time First (SSTF)
- SCAN (Elevator)
- C-SCAN
- LOOK

📊 Output
- Execution order
- Total seek time
- Visual movement of disk head

🌐 Live Demo

🛠 Tech Stack
- HTML
- CSS
- JavaScript

📌 Purpose
Built to demonstrate OS disk scheduling concepts visually.
